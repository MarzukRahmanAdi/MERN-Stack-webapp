# installation (frontend)
```
cd client
npm install 
or
yarn
```

# installation (backend)
```

cd server
npm install 
or
yarn

## then add a .env file like this 
MONGODBURI=Your Link

```

# Run
```

## backend
1. cd server
2. yarn run dev


## front-end
1.Go to /src/api/index.js file and set it to localhost:5000
2.Run on terminal -  yarn start
3. Go to localhost:3000

```
